# ✅ VerifyLine — Phone Validation API


**VerifyLine** is a lightweight phone validation service that allows users to validate international phone numbers, detect carrier and region information, and track API usage against a request limit.

## ✨ Features 🌟

### 👤 For Users 🧑‍💻
- 📝 Registration with auto-generated API keys
- 🔑 Secure login system
- 📞 Validate international phone numbers
- 📡 Detect carrier & region info
- 📊 Track usage against limit
- 📋 View request history

----
### 🎨 User Interface
- 🪟 Modern glassmorphism design
- 🌙 Full dark/light mode support
